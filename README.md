# PersonalEnv

Personal bash environment set up with familiar shortcuts.

## Setup 

```
$ cd ~
$ git clone https://github.com/crdoconnor/personalenv
$ ./personalenv/conf/setup.sh
```

## Teardown

```
$ cd ~
$ ./personalenv/conf/unsetup.sh
$ rm -rf ./personalenv
```
